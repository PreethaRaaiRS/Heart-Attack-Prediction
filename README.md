A Machine Learning project developed to predict the likelihood of a heart attack using clinical health data and classification algorithms.

The system includes data preprocessing, feature selection, model training, and optimization using techniques such as Ant Colony Optimization (ACO) and Genetic Algorithm. Model performance is evaluated using metrics like Accuracy, Precision, Recall, and F1-Score.

This project was developed as a collaborative academic project to apply machine learning concepts to real-world healthcare data.
