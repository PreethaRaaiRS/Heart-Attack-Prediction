# 🫀 Heart Attack Prediction using Machine Learning

## Project Overview

Heart disease is one of the leading causes of death worldwide. This project focuses on building a Machine Learning model to predict the likelihood of a heart attack based on clinical and health-related features.

The model analyzes patient information such as age, cholesterol level, resting blood pressure, maximum heart rate, and other medical attributes to classify whether a person is at risk of a heart attack.

---

## Objective

* Perform data preprocessing and cleaning
* Train classification models for prediction
* Optimize model performance using meta-heuristic algorithms
* Evaluate the model using standard performance metrics

---

## Features

* Data cleaning and preprocessing
* Feature selection
* Model training and evaluation
* Optimization using Ant Colony Optimization (ACO) and Genetic Algorithm
* Model saving for future predictions
